# zoho-web-2-leads
Zoho CRM API 1 Web 2 Leads

Simple PHP script to insert a new Leads record in Zoho CRM using API v1
It returns a blank gif (perfect for your email campains)

How to use it:
yourserver/insert.php?
&company=COMPANY
&ln=LASTNAME
&fn=FIRSTNAME
&email=EMAIL
&phone=PHONE
&source=SOURCE
&website=WEBSITE
&city=CITY
&industry=INDUSTRY
&lo=LEAD OWNER
&designation=DESIGNATION
&fax=FAX
&mobile=MOBILE
&ls=LEAD STATUT
&ne=50
&ar=99999999
&eoo=True
&skype=SKYPE
&salutation=MRS 
&street=STRRET
&state=STATE
&zip=ZIP 
&country=COUNTRY
&description=DESCRIPTION
&rating=Active
&semail=SECONDARY EMAIL
&title=TITLE
